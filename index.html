Scratch Game

Riley's Kitchen Adventure is a game that I created using Scratch, along with a friend, for a video game design class at Vanier Cegep. This game tells the story of a mouse named Riley, whose goal is to collect fruit in order to unlock barriers and reach the cheese to advance to the next level.

To add a layer of difficulty, there is a time limit and a health bar. The health bar decreases each time the player hits a fork, and it can be regained by collecting fruit. Clocks are also included in the game, which grant the player more time. The game features background music and sound effects, which enhances the overall gaming experience.

The game can be played by moving the mouse using the arrow keys on the keyboard. The player can jump and move up, down, left, and right to navigate through the game.

Riley's Kitchen Adventure is a significant project for me as it was the first project I ever created, and it marked the beginning of my coding journey. This project gave me the opportunity to apply my creativity and problem-solving skills to design a fun and engaging game. It also taught me the basics of game development, and it has inspired me to continue exploring new ideas and creating exciting projects.
